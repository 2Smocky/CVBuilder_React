# Manual de Uso - CV Designer

## 1. Introducción

Bienvenido al **CV Designer**. Esta aplicación te permite crear, personalizar y descargar tu currículum vitae de una manera sencilla e intuitiva. Con una interfaz amigable, puedes rellenar tus datos y ver los cambios en tiempo real sobre una plantilla profesional.

## 2. Requisitos Previos

Para utilizar la aplicación, solo necesitas un navegador web moderno como:
*   Google Chrome
*   Mozilla Firefox
*   Microsoft Edge
*   Safari

No se requiere ninguna instalación.

## 3. Guía de Inicio Rápido

Sigue estos pasos para crear tu CV:

1.  **Rellena tus datos:** Utiliza el formulario de la izquierda para introducir tu información personal, experiencia, educación, habilidades, etc.

2.  **Visualiza en tiempo real:** A medida que escribes, el panel de la derecha se actualizará automáticamente, mostrándote cómo quedará tu CV.

3.  **Cambia de plantilla (Opcional):** Si hay varias plantillas disponibles, puedes seleccionarlas para ver cuál se adapta mejor a tu perfil.

4.  **Descarga tu CV:** Cuando estés satisfecho con el resultado, haz clic en el botón **"Descargar PDF"**. Tu currículum se guardará como un archivo PDF en tu dispositivo.

## 4. Funcionalidades Principales

*   **Edición Intuitiva:** Un formulario claro y dividido por secciones para facilitar la entrada de datos.
*   **Previsualización Instantánea:** Observa los cambios al momento sin necesidad de recargar la página.
*   **Exportación a PDF:** Descarga una copia de alta calidad de tu CV con un solo clic.
